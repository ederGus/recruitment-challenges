He intentado, dentro de lo que cabe, que la demo cumpla con las expectativas pedidas en el reto.

En principio, habia pensado hacer un proyecto para front-end y otro para backend(web-api en .net), pero al leer los requisitos y ver que uds. proporcionaban los endpoints no le veía mucho sentido a montar el back-end project.

He clonado(git) el proyecto y he creado una feature branch llamada "EderChallengeFullStack", he subido todo el proyecto al repositorio proporcionado.

A pesar de disponer de un tiempo más que considerable para la entrega de del reto, no he incluído ciertas funcionalidades que me hubieran gustado añadirlas como que la aplicación esté en dos idiomas, un interceptor para el manejo de errores, crear unos cuantos test, etc.; no me ha dado tiempo por motivos personales a realizar lo mencionado antes

Para construir este proyecto he utilizado las siguientes tecnologías:

  • Nodejs
  • Angular 5
  • TypeScript
  • Bootstrap 3
  • ...

He enviado por correo(jobs.spain@payvision.com) el proyecto empaquetado(carpeta dist) si se quiere ver la aplicacion en un navegador web podemos hacerlo de las siguientes maneras:.

Hay que descomprimir la carpeta.
  • Subir el contenido del proyecto empaquetado(carpeta dist) a un servidor web
  • Clonar el proyecto que está en el repositorio(https://github.com/ederGus/recruitment-challenges/tree/EderChallengeFullStack)
    • Descargar e instalar node(https://nodejs.org/es/download/)
    • Después de instalar node, ejecutamos el siguiente comando npm install @angular/cli -g
    • Nos situamos a la altura de la carpeta transactions-payvision, y ejecutamos el comando npm install
    • Por último, arrancamos el proyecto para visualizarlo en una navegador web, ejecutamos el siguiente comando ng serve --open
